# Arcadia 
Arcadia est une application web permettant de gérer les habitats, services, et animaux d’un zoo. 
Elle offre des fonctionnalités pour différents rôles :
- **Admin :** Gestion des utilisateurs, services, habitats, et statistiques.
- **Employé :** Validation des avis et gestion des rations alimentaires.
- **Vétérinaire :** Ajout de comptes rendus sur les animaux.

## Installation
